# FlappyRandomizer  
**Link: https://frlandek.github.io/Games/flappyrandomizer/FlappyRandomizer.html**  

**Flappy Bird, but randomized!**  
Things that are random:  
1. Background Type  
2. Pipe Color
3. Bird Color
4. Bird Size
5. Bird Jump Height

Future Updates:

2. Pipe Opening Randomizer
3. Bird Rotation

5. Make it so you can't fly over pipes
